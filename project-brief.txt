Project: AI-Powered Proposal Writer

Goal:
Build a web app that helps consultants generate, edit, and export professional proposals.

Must Have:
- User authentication with role-safe data access
- Proposal wizard with document upload, AI analysis, and final export
- Pricing per deliverable and milestone payment schedule
- Dashboard for managing proposals

Technical Expectations:
- React frontend with responsive UI
- Node.js API with MongoDB
- AI generation with structured JSON output
- PDF export with clear sections and formatting

Delivery Constraints:
- Clear validation and user-friendly errors
- Maintainable code and modular structure
- End-to-end workflow should work without manual DB edits
